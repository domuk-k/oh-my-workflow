## 지금 상태 (DONE — green)

TDD로 빌드한 **하중 지지 코어**. `bun test` 66 pass / 0 fail, `tsc --noEmit` clean, 라인 커버리지 ~99.7% (runtime/journal/fake 100%, schema-gate 99%).

- `src/runtime.ts` — `makeRuntime` 5훅(`agent`/`pipeline`/`parallel`/`phase`/`log`), null-contract, 동시성 limiter(기본 4, **정석 세마포어** — TOCTOU 레이스 수정 완료).
- `src/schema-gate.ts` — `extractJson`(결정적 우선순위: 마지막 fenced 블록 → 최대 balanced-brace), `makeValidator`(ajv), `schemaGate`(≤2 재시도 → null, never throws).
- `src/journal.ts` — JSONL 이벤트, `promptHash`/`optsHash`/`resumeKey`(undefined 키 스킵 → resume 키 안정), 시계 주입.
- `src/adapters/types.ts` — `AgentPort` 계약.
- `src/adapters/fake.ts` — 결정적 fake 어댑터(테스트 더블 + 곧 `--agent fake`). 시퀀스 응답 + fault injection + sessionId 라우팅.
- 테스트: `test/*.test.ts` 7개 파일. 게이트 = `test/spine.test.ts`.

### 락된 계약 (v2가 포맷 변경 없이 얹히도록 — 절대 깨지 말 것)
- **null-contract**: `agent()`는 절대 throw 안 함. 종료 실패 = `null` + `agent_end{ok:false, kind}`. `kind` ∈ adapter(`timeout`/`nonzero_exit`/`spawn_failure`) | gate(`no_json`/`schema_violation`) | **`internal_error`**(우리 코드 버그 — 어댑터 실패와 구분). 진단 페이로드 `stderr`/`rawText`/`error`도 저널에 실림.
- **resume 키** = `(callIndex, promptHash, optsHash)`. CC의 최장-불변-접두사 모델. 바이트 안정 검증됨.
- **저널 이벤트** = run_start/phase/agent_start/attempt/agent_end/log/run_end. 페어링(고아 없음) 검증됨.
- **schema-gate 재시도** = 정확히 2회(총 3 호출). 어댑터 ok:false는 재시도 없이 단락. 작성 에이전트는 스키마 노이즈를 안 봄.

## 백로그 (우선순위순)

### P0 — MVP 완성 (출시 전 필수)
1. **CLI 러너** `src/cli/run.ts` (+ `package.json` bin은 이미 `omw`):
   - `omw run wf.js --agent <a> --args '{...}' [--concurrency N] [--pretty]`
   - stdout = **결과 JSON 한 덩어리만**. 저널 JSONL은 `.omw/r-<id>.jsonl`. `--pretty`는 phase/fan-out 트리를 **stderr**로.
   - 워크플로우 모듈 로드 → `makeRuntime` 주입 → `journal.runStart`/`runEnd` 브래킷 → 실행 → 결과 직렬화.
   - 종료코드: 0 완주 / 1 스크립트 에러(throw·문법, stderr에 구조화 JSON) / 2 사용법 / 3 어댑터 부재(`{"error":"adapter_missing","install_hint":…}`).
2. **`--agent fake` 배선** — `src/adapters/fake.ts`를 CLI에서 선택 가능하게. **이게 README 최상단 무료 try-it 라인이자 demo-ability FATAL의 해소책.** `examples/deep-research`가 `--agent fake`로 결정적으로 green 나야 함.
3. **`src/cli/replay.ts`** — `omw replay .omw/r-<id>.jsonl [--json]`. 저널에서 phase/fan-out/stats 재구성. **fixture replay로 정직하게 라벨**(라이브 resume 아님).
4. **`examples/deep-research/`** — workflow-anatomy 척추 이식. `--agent fake`(무료·결정적) + `--agent claude`(OMW_LIVE 뒤 실호출 e2e) 둘 다.
5. **`adapters/claude.ts`** — `claude -p --output-format json`, `.result` 파싱, **필드 리네임 golden fixture**(`session_id→sessionId`, `total_cost_usd→costUsd`, `duration_ms→durationMs`, `is_error/subtype→{ok:false}`). `--resume`로 followUp. (claude 2.1.177 로컬 확인됨.)

### P1 — SKILL (주 진입점) + 정직 표면
6. **`skill/SKILL.md`** + 패턴 템플릿: fan-out / verify-vote(**기권 정족수** 포함) / pipeline / loop-until-dry. run→journal→fix 디버그 루프 섹션. 관례 섹션(비결정성 회피, null-contract 위 작성, 기본 `filter(Boolean)`).
   - **B3 측정**: SKILL.md만으로 호스트 에이전트가 3개 fresh task ×10 trial 중 ≥8 runnable 스크립트 나올 때까지 반복.
7. **`oh-my-workflow doctor`** — 각 어댑터 `--version`+auth 프로브, v1 adapter matrix 출력.
8. **README 출시판** — `--agent fake` above the fold, adapter matrix(claude full / codex,pi experimental) + `openai/codex#15451` caveat + "no sandbox/결정성=관례" 노트를 **첫 화면**에(B12). 전략 문서 §2의 **락된 정직 언어**를 그대로 쓸 것 — 과대주장(agent-authored=런타임속성, "CC 5훅 검증", deterministic) 재유입 금지.

### P2 — Substantial 바 (멀티어댑터 + 벤치)
9. **`adapters/codex.ts`** — `codex exec --json` JSONL 최종 메시지 추출. **#15451 malformed YAML 관용 추출기**(조용히 실패 말고 행동가능하게 실패). experimental 라벨. (codex 0.137.0 로컬 확인됨.)
10. **`adapters/pi.ts`** — `pi --print`. **pi 로컬 미설치** → 실측 필요(설치 후). experimental.
11. **벤치마크 하니스** — fake-adapter 오버헤드 격리(per-call ms, % wallclock, journal bytes/call, retry rate). 라이브 숫자는 "indicative, non-reproducible" 라벨.
12. **CI** — 매 push: `bun test` + `tsc` + 커버리지 게이트(≥90% 순수부) + **golden-journal 스냅샷 diff**(B7: 커밋된 골든 저널이 스키마/해싱 drift에 깨지게). 3환경 매트릭스(macOS/ubuntu/devcontainer) fake e2e + nightly OMW_LIVE claude smoke.

### P3 — 서사 + 출시
13. **Category 에세이/해부 블로그** — 오너 노트 앵커(LLM=센서·스크립트=컨트롤러, decomposition-enables-delegation, KR '시멘트 붓기' 은유). 4-way 포지셔닝 표(Bernstein/pi-builder/sub-agents-skills/CC Workflow). workflow-anatomy 정직 ledger("재구성이지 바이너리 해부 아님").
14. **asciinema/agg GIF** — `--agent fake` run + 자가수정 루프(무료·즉시·재현).
15. **출시 옵스** — `npm publish --provenance` as `oh-my-workflow`, zce에 `omw` 이름 transfer 요청, skills.sh 제출(public repo 먼저). 블로그 애널리틱스(scroll+outbound) + HN Algolia/GN-vote 폴링 cron **사전 계측**. Tue–Thu ~8–10am PT(또는 일요일 저경쟁). firstComment 즉시, 부스터 코멘트 금지, 2–3h 베이비싯(전략 §6 레드팀 반박 키트).

### 코어 정리(작은 빚 — 리뷰에서 연기됨)
16. **`extractJson` no_json vs invalid_json 구분** — 현재 둘 다 `no_json`. 후보(펜스/brace)는 있었는데 파싱 실패한 경우 `invalid_json`으로 분리하면 자가수정 정확도↑. (지금은 `rawText`가 저널에 실려 90% 보완됨.)
17. **펜스 내 ` ``` ` 견고성** — 문자열 안 트리플백틱이면 펜스 정규식이 조기 종료. **현재 balanced-brace 폴백으로 복구됨**(결정적). characterization 테스트 추가하고, 필요시 펜스 스캐너를 상태머신으로.
